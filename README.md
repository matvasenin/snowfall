# О проекте
***Snowfall*** — это *фреймворк*, целью которого является обеспечение безопасности *MCP-сервера* и/или *MCP-клиента*.

# Преимущества
- ***Расширяемость***: *гибкая система плагинов* позволяет реализовывать дополнительный функционал за считанные минуты
- ***Прозрачность***: Snowfall *незаметен* и для пользователя, и для злоумышленника

# Структура
- ***/assets*** — статические файлы, в т.ч. логотипы
- ***/src*** — исходный код
  - ***/models*** — глобальные структуры
  - ***/plugins*** — базовые плагины
  - ***/i18n*** — файлы локализации
